# Add your photos here

Place your birthday celebration photos in this directory. Supported formats:
- JPG / JPEG
- PNG
- WebP (recommended for best compression)

File naming convention:
- photo-1.jpg
- photo-2.jpg
- etc.

Recommended specifications:
- Resolution: 1200x1200px or higher
- File size: 200-500KB each
- Format: JPG for photos, PNG for images with transparency
- Aspect ratio: Square or landscape (auto-crops to square in grid)

The website supports 6-12 photos for the best visual impact.
