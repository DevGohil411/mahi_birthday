## 💡 Key Features Explained

### Hero Section (3D + Parallax)
- **What**: Fullscreen section with rotating 3D object
- **Why**: Creates impressive, memorable first impression
- **How**: Uses React Three Fiber for 3D, Framer Motion for parallax

### Photo Gallery (Responsive Grid)
- **What**: Beautiful photo grid with hover effects
- **Why**: Showcases photos with elegant interactions
- **How**: Tailwind grid layout, Framer Motion hover animations

### Highlight Strip (Horizontal Scroll)
- **What**: Magazine-style horizontal scrolling gallery
- **Why**: Adds visual interest and unique interaction
- **How**: CSS flexbox with scroll container

### Message Section (Text Reveal)
- **What**: Emotional text with line-by-line animation
- **Why**: Delivers message with impact
- **How**: Framer Motion stagger animation

### Finale (Confetti + CTA)
- **What**: Closing section with celebration confetti
- **Why**: Creates joyful ending moment
- **How**: Canvas-based lightweight animation
